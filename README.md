# Servionics Demo Website

Demo-Website für Servionics GmbH - Robot-as-a-Service für den Mittelstand.

## Live Demo

Diese Website ist als GitHub Pages gehostet und kann direkt im Browser angezeigt werden.

## Lokale Verwendung

1. Repository klonen
2. `index.html` in einem Browser öffnen

## Struktur

- `index.html` - Haupt-Website mit integriertem CSS und JavaScript
- `image_8.png` - Prozessgrafik

## Technologie

- HTML5
- CSS3 (mit CSS Custom Properties)
- Vanilla JavaScript
- Responsive Design
